# Animal Crossing Villager Index

An online application used to search Animal Crossing villagers and find their statistics. 

### Technologies Used

- CSS
- HTML
- JavaScript
- jQuery
- Google Fonts
- Nookipedia API: https://api.nookipedia.com/doc
